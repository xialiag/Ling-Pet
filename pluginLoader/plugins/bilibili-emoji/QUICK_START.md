# B站表情包插件 - 快速开始

## 🚀 5分钟上手

### 1. 安装插件

插件会自动安装到用户数据目录。

### 2. 下载表情包

```
用户: "帮我下载鸽宝表情包"

LLM 会自动:
1. 搜索B站装扮
2. 显示搜索结果
3. 下载表情包
4. 回复下载完成
```

### 3. 发送表情包

```
用户: "发个开心的表情"

LLM 会自动:
1. 搜索本地表情包
2. 选择合适的表情
3. 在回复中显示表情包
```

## 📋 LLM 工具速查

| 工具 | 功能 |
|------|------|
| `search_local_emoji("开心")` | 搜索本地 |
| `search_bilibili_emoji("鸽宝")` | 搜索B站 |
| `download_emoji_suit(id, type)` | 下载装扮 |
| `send_emoji("开心")` | 发送表情 |
| `random_emoji()` | 随机表情 |
| `list_emoji_categories()` | 列出分类 |
| `rescan_emojis()` | 重新扫描 |

## 🎯 常用场景

### 下载新表情包
```
用户: "帮我找一些可爱的表情包"

```

### 发送表情包
```
用户: "发个开心的表情"
→ LLM 搜索并发送
```

```
用户: "来个随机表情包"

```

```
用户: "我有哪些表情包分类"

```
